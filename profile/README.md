## Features 🚀
* Auth
  - [x] Register
  - [x] Login
  - [x] Forgot Password
* Profile
  - [x] Activity
  - [x] Settings:
    - [x] Edit Profile
    - [x] Change Password
  - [x] Logout
* Course
  - [x] Lessons
  - [x] Exercises
  - [x] Exams
